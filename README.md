# LeetCode Algorithms (C++)

这个仓库收集了我在 LeetCode 平台的题解。每道题以单文件形式保存，文件名使用“题号.题目名.cpp”的格式，便于检索和复盘。

## 仓库结构

```text
leetcode/
├── algorithms/
│   ├── 1.两数之和.cpp
│   ├── 11.盛最多水的容器.cpp
│   ├── ...
└── README.md
```

## 贡献指南

欢迎提交Pull Request来贡献新的题解或优化现有解法。请确保:

1. 代码风格一致且有详细注释
2. 提供清晰的解题思路
3. 如果有多种解法，请尽量都包含在内并比较它们的优缺点
